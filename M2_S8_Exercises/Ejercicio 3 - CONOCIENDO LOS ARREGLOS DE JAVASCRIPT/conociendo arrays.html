Conociendo Los Arrays

- QUE ES UN ARRAY (Arreglo)
- LENGTH : CONSEGUIR EL LARGO DE UN ARREGLO
- ACCEDER A UN ELEMENTO DEL ARREGLO POR SU INDICE
- RECORRER UN ARREGLO UTILIZANDO FOR
- FOR OF
- UNSHIFT : AÑADIR ELEMENTOS AL INICIO DEL ARREGLO
- SHIFT : ELIMINAR ELEMENTOS DEL INICIO DE UN ARREGLO
- PUSH : AÑADIR ELEMENTOS AL FINAL DEL ARREGLO
- POP : ELIMINAR ELEMENTOS DEL FINAL DEL ARREGLO
- INDEX OF : ENCONTRAR EL INDICE DE UN ELEMENTO DENTRO DEL ARREGLO
- SPLICE : ELIMINAR ELEMENTOS MEDIANTE SU POSICIÓN
- CREANDO UN ARRAY A PARTIR DE ELEMENTOS ELIMINADOS : SPLICE
- FILTER : FILTRA ELEMENTOS DE UNA RRAY , Y LOS EXTRAE EN UN NUEVO ARRAY
